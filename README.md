# 🏦 Loan Default Prediction System

A comprehensive machine learning system for predicting loan default risk using advanced ensemble methods.

##  Project Structure

```
Loan_Default_Project/
├── advanced_loan_default.py      # Model training script
├── app.py                         # Flask API server
├── dashboard.html                 # Web dashboard with prediction form
├── loan_data.csv                  # Training dataset
├── requirements.txt               # Python dependencies
├── advanced_loan_default_model.pkl # Trained model (generated after running training)
├── roc_curve.png                 # ROC curve visualization (generated)
├── feature_importance.png        # Feature importance chart (generated)
└── templates/                    # Flask templates directory
```
